## How It Works

- **OpenClaw** manages all notes as Markdown files
- Notes are committed to GitHub: [jccollinsdev/knowledge-base](https://github.com/jccollinsdev/knowledge-base)
- Obsidian on any device syncs via Git
- Secrets are stored in secrets manager, not in notes
